🤖 Auto-Reward Top Remixers Weekly (GTT or Stripe)
1. Cron Script (Replit or GitHub Actions)
ts
Copy
Edit
// /scripts/rewardTopRemixers.ts
const { data: topRemixers } = await supabase
  .from("remix_contest_entries")
  .select("remix_id, user_id, votes")
  .eq("contest_id", activeContestId)
  .order("votes", { ascending: false })
  .limit(3);

// Reward logic
for (const remix of topRemixers) {
  await supabase.from("remix_earnings").insert({
    remix_id: remix.remix_id,
    user_id: remix.user_id,
    earnings: 10.0,
    payout_tx: await sendGTT(remix.user_id, 10.0)
  });

  await supabase.rpc('increment_user_remix_awards', { user_id: remix.user_id });
}
2. Replit Scheduler (Cron)
sh
Copy
Edit
*/10 * * * * node scripts/rewardTopRemixers.ts
