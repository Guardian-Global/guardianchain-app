## <span id="faq-list">Frequently Asked Questions <ListIcon /></span>

<FAQ>
  <Question q="What is GuardianChain?">
    A protocol for sovereign digital memory and truth—see the Protocol
    Whitepaper.
  </Question>
  <Question q="How do I mint a capsule?">
    Connect your wallet and follow the onboarding steps. See
    Validator/Onboarding Guide.
  </Question>
  <Question q="What is GTT?">
    The native, fixed-supply token for protocol utility, rewards, and
    governance. See Tokenomics.
  </Question>
  <Question q="How do I participate in governance?">
    Hold GTT, connect, and vote/propose via the DAO portal.
  </Question>
  <Question q="How do I run a validator?">
    See Validator Guide or try the onboarding wizard.
  </Question>
  <Question q="How do I get technical support?">
    Ask the AI, join community chat, or escalate to the DAO support team.
  </Question>

    All docs, audits, and legal opinions are downloadable from Compliance page.
  </Question>
  <Question q="Where can I find your API or SDKs?">
    See Developer Portal for live docs, code samples, and OpenAPI explorer.
  </Question>
  <Question q="How can I report bugs or security issues?">
    Submit via the Security Bounty section or open a DAO proposal.
  </Question>
  <Question q="Can I help translate or improve docs?">
    Yes, every page supports DAO/community annotation, translation, and
    upvoting.
  </Question>
